#### 开启adb

- 将Android sdk目录加到环境变量

``` shell
echo '' >> ~/.zshrc
echo 'export ANDROID_HOME=/Users/$USER/Library/Android/sdk' >> ~/.zshrc
echo 'export PATH=$PATH:$ANDROID_HOME/platform-tools' >> ~/.zshrc
source ~/.zshrc

```

#### 安卓手机

- 先用数据线连上Android studio
- 使用adb命令连接：``` adb connect 手机IP:5555 ```
